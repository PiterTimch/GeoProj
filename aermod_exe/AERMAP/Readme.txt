CO (Control Options)

TITLEONE — назва запуску.

DATATYPE — тип DEM-файлів (DEM1 = USGS).

DATAFILE — список DEM-файлів для рельєфу.

TERRHGTS — спосіб отримання висот рельєфу (PROVIDED = задано вручну).

DOMAINXY — координати області (xmin, ymin, zone, xmax, ymax, zone).

ANCHORXY — точка прив’язки для координат UTM.

RUNORNOT — запускати розрахунок чи ні.


SO (Source Locations)

LOCATION — визначає джерело: ім’я, тип (POINT), координати X/Y.


RE (Receptors)

DISCCART — координати рецептора (X, Y) та висота Z.



OU (Output Options)

RECEPTOR — вихідний файл з рецепторами.

SOURCLOC — вихідний файл з джерелами.
