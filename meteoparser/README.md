## дані Era5
```
ERA5 Surface (поверхневі дані):
t2m — температура повітря на висоті 2 м, одиниці: Кельвіни (K); після обробки: temperature_2m_C = t2m - 273.15, одиниці: °C.
u10 — компонент вітру у напрямку захід–схід на висоті 10 м, одиниці: м/с.
v10 — компонент вітру у напрямку північ–південь на висоті 10 м, одиниці: м/с.
wind_speed_10m — швидкість вітру, обчислюється як sqrt(u10² + v10²), одиниці: м/с.
wind_dir_10m — напрям вітру (азимут від півночі), обчислюється як (180 + atan2(u10, v10)) % 360, одиниці: градуси.
msl — атмосферний тиск на рівні моря, одиниці: Паскалі (Pa); після обробки pressure_hPa = msl / 100, одиниці: гектопаскалі (hPa).
tcc — загальна хмарність, одиниці: частка неба (0–1).
ssrd — сумарна сонячна радіація, одиниці: Дж/м²; можна перевести у Вт/м², поділивши на 3600.
tp — сумарні опади, одиниці: метри; для зручності можна перевести у мм (tp * 1000).


ERA5 Upper-Air (висотні дані):
t — температура повітря, одиниці: Кельвіни (K); після обробки temperature_C = t - 273.15, одиниці: °C.
u — компонент вітру у напрямку захід–схід, одиниці: м/с.
v — компонент вітру у напрямку північ–південь, одиниці: м/с.
wind_speed — швидкість вітру, обчислюється як sqrt(u² + v²), одиниці: м/с.
wind_dir — напрям вітру (азимут від півночі), обчислюється як (180 + atan2(u, v)) % 360, одиниці: градуси.
z — геопотенціальна висота, одиниці: м²/с²; після обробки height_m = z / 9.80665, одиниці: метри.
pressure_level — стандартний рівень тиску (1000, 925, 850), одиниці: гектопаскалі (hPa).
```
